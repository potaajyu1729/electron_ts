electron_ts セットアップ手順（Windows）

----------------------------------------
【1】必要なもの
----------------------------------------

・Windows 10 または 11 のパソコン
・USBメモリ
・インターネット接続

----------------------------------------
【2】Node.js をインストールする
----------------------------------------

① 以下のページを開く
https://nodejs.org/ja

② 「LTS」と書かれた方をダウンロード
③ ダウンロードしたファイルを開き「次へ」を押してインストール

④ 確認
スタートメニューで「PowerShell」を開き、次を入力：

node -v

数字（例：v20.15.1）が出ればOK

----------------------------------------
【3】このプロジェクトを準備する
----------------------------------------

① GitHubからZIPをダウンロード
・「Code」→「Download ZIP」
・右クリックして「展開」

② 展開したフォルダを開く
（electron_ts フォルダ）

----------------------------------------
【4】必要なライブラリを入れる
----------------------------------------

electron_ts フォルダ内で
PowerShell / コマンドプロンプトを開き、次を実行：

npm install

----------------------------------------
【5】USBに openapp フォルダをコピー
----------------------------------------

① GitHubにある openapp フォルダをUSBの直下にコピー
例：E:\openapp\index.html

----------------------------------------
【6】アプリを起動する
----------------------------------------

PowerShell / コマンドプロンプトで以下を実行：

npm start

・ウィンドウが開く
・「USBを挿してください」と表示される

----------------------------------------
【7】USBを挿す
----------------------------------------

USBメモリを挿すと、自動で
USB内の openapp/index.html が表示される

----------------------------------------
【8】うまくいかないとき
----------------------------------------

・USBに openapp フォルダが入っているか確認
・USBを挿している状態でアプリを起動しているか確認
・エラーが出る場合はアプリを再起動してUSBを再挿入

----------------------------------------
【9】補足
----------------------------------------

- 起動時は空ページで待機
- 本体にはHTMLは不要
- 複数USBがある場合は最新で挿入されたUSBを表示
